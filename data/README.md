# Dataset Information

## Source
**Brazilian E-Commerce Public Dataset by Olist**  
📎 [Download from Kaggle](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)

## Overview
This dataset contains ~100,000 orders placed on the Olist e-commerce 
platform between 2016–2018 across multiple Brazilian marketplaces.

## Tables Used in This Project
| Table | Description |
|---|---|
| olist_orders_dataset | Order status, timestamps |
| olist_order_items_dataset | Items per order, price, freight |
| olist_customers_dataset | Customer location and ID |
| olist_order_payments_dataset | Payment type and value |

## Note
Raw CSV files are not included in this repository due to file size.  
Please download directly from Kaggle using the link above.
